博客迁移

2020-09-04
Word count: 516 | Reading time: 2min

前言

由于博客是搭建在公司电脑上的，今天有时间迁移到自己的电脑。

前期准备

在新的电脑安装：

  • 安装git
  • 安装Node.js
    Git用于把部署在GitHub的内容获取至本地，Node.js用于安装Hexo。
    如果电脑已经安装Git和Node.js，那么只需要用npm即可完成Hexo的安装： //部署整个站点
    然后输入
    git config –global user.email “你的邮箱”
    git config –global user.name “你的名字”
    （注意 “ 前面是有空格的）
    期间还遇到了ssh密钥的问题，好像也不影响。这样就大功告成了。
